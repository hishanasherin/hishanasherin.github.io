## 🌐 Deploy on GitHub Pages (Free)

1. Go to [github.com](https://github.com) → **New Repository**
2. Name it: `hishanasherin.github.io` → **Public** → **Create**
3. Click **Add file → Upload files** → upload all files keeping folder structure
4. Go to **Settings → Pages → Branch: main → / (root) → Save**
5. Live at: `https://hishanasherin.github.io` ✅
